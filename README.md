# 🎯 Seaborn Tips Streamlit Dashboard

🚀 An interactive and visually appealing **Data Visualization Web App** built using **Streamlit, Seaborn, and Matplotlib** to explore restaurant tipping behavior from the famous **Tips Dataset**.

---

## 🌟 Live Project Overview

This project transforms raw restaurant billing data into **interactive insights** using beautiful visualizations. It helps understand how factors like bill amount, table size, gender, and day affect tipping behavior.

👉 Built for learning, practice, and showcasing Data Visualization skills.

---

## 📊 Key Features

✨ Interactive Streamlit Dashboard  
📌 Scatter Plot → Total Bill vs Tip (with time & size)  
📈 Line Plot → Table Size vs Total Bill (gender-wise)  
📊 Bar Plot → Day vs Total Bill (gender comparison)  
🎨 Clean and modern Seaborn visualizations  
⚡ Fast and interactive UI

## 📸 Dashboard Visualizations

### 🔹 Scatter Plot
Shows relationship between total bill and tip with additional insights like time and group size.

### 🔹 Line Plot
Displays how total bill varies with group size and gender.

### 🔹 Bar Plot
Compares total bill across different days and gender groups.
